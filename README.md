# o-label [![Build Status](https://secure.travis-ci.org/Archetype-CSS/o-label.png?branch=master)](http://travis-ci.org/Archetype-CSS/o-label) [![Built with Grunt](https://cdn.gruntjs.com/builtwith.png)](http://gruntjs.com/)

Archetype object for building label components.

## Installation
  * [Bower](http://bower.io): {{coming soon}}
  * Git: `git clone https://github.com/Archetype-CSS/o-label.git`

## Use
  * `o-label` - object styles for building label components

## Run the Test Locally

```bash
git clone https://github.com/Archetype-CSS/o-label.git
cd o-label
npm install
grunt
```
